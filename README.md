# Video Player

Custom Video Player

Check the demo hosted on https://filipeqs.github.io/video-player/

---

## Instructions

First clone this repository

```bash
$ git clone https://github.com/filipeqs/video-player.git
```

Open index.html

```bash
open index.html
```

## Features

- Play/Pause Video
- Mute/Unmute Video
- Play Video on Fullscreen
